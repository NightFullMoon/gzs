## Build Setup/使用说明

[查看中文demo](http://meishenmeyong.com/gzs/)

``` bash
# install dependencies / 安装依赖的项目
npm install

# serve with hot reload at localhost:8080 / 然后运行，服务器地址：localhost:8080，附带着热重载功能
npm run dev

# build for production with minification / 构建带压缩的产品
npm run build
```
