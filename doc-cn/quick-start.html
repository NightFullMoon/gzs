快速开始 / Quick-start

使用webpack打包【推荐】

首先，将项目下载到本地 ，运行命令

npm install
npm run gzs

将dist目录下的所有文件复制一份到你的目录下

import gzs from "./gzs.js"
Vue.use(gzs);

就可以开始使用了

使用传统方式

像往常一样引入js和css文件：

<script type="text/javascript" src="http://meishenmeyong.com/gzs/dist/gzs.js"></script>
<link rel="stylesheet" type="text/css" href="http://meishenmeyong.com/gzs/dist/gzs.min.css">

就可以开始使用了
